* Description
Der neue Küchenschrank der fleißigen Oma nach dem Umzug von Vanilla-Emacs nach Spacemacs.
Mit vielen gut erhaltenen und jetzt renovierten Utensilien aus dem alten Küchenschrak.
Dazu kommen neue, moderne Geräte. 
Alles aber hoffentlich im neuen Spacemacs-Stil organisiert.

** Features:
  - Autocomplete
  - Lint
  - Refactor
  - ...

* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =asl_küchenschrank= to the existing =dotspacemacs-configuration-layers= list in this
file.
